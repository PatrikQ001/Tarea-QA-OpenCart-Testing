# QA-OpenCart-Testing

Proyecto de **Pruebas de Software** enfocado en el diseño, implementación y ejecución de **pruebas unitarias** sobre módulos críticos de OpenCart utilizando **PHPUnit**, **GitHub** y **GitHub Projects**.

---

## Descripción

El objetivo de este proyecto es evaluar la calidad, estabilidad y confiabilidad de OpenCart mediante la aplicación de pruebas unitarias a componentes clave del sistema.

Los módulos seleccionados para el análisis son:

- Login y Registro
- Carrito de Compras
- Checkout / Pago
- Catálogo y Búsqueda
- Gestión de Inventario
- Sistema de Reseñas

---

## Objetivos

- Validar el correcto funcionamiento de los módulos seleccionados.
- Detectar errores en etapas tempranas del desarrollo.
- Aplicar buenas prácticas de aseguramiento de la calidad del software.
- Automatizar la ejecución de pruebas mediante PHPUnit.
- Mejorar la estabilidad y mantenibilidad del sistema.
